# VR Photo Practice

[![Build Status](https://travis-ci.org/miya0001/vrphoto.svg)](https://travis-ci.org/miya0001/vrphoto)

## How to build

```
$ git clone git@github.com:miya0001/vrphoto.git
$ cd vrphoto
$ npm install
$ npm run build
$ npm start
```

### Build requires

* imagemagick
* graphicsmagick
